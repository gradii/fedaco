# Fedaco ORM

Laravel Eloquent-style ORM for TypeScript and Node.js.

Fedaco brings the expressiveness and convenience of Eloquent's Active Record pattern to the TypeScript ecosystem, with full support for relationships, migrations, query building, and connection pooling.

## Installation

```bash
npm install @gradii/fedaco
# or
pnpm add @gradii/fedaco
# or
yarn add @gradii/fedaco
```

You also need a database driver:

| Database   | Driver Package                    |
|------------|-----------------------------------|
| MySQL      | `@gradii/fedaco-mysql-driver`     |
| PostgreSQL | `@gradii/fedaco-postgres-driver`  |
| SQLite     | `@gradii/fedaco-sqlite-driver`    |
| SQL Server | `@gradii/fedaco-sqlserver-driver` |
