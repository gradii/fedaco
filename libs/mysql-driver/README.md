# @gradii/fedaco-mysql-driver

MySQL driver for [Fedaco ORM](https://www.npmjs.com/package/@gradii/fedaco). Provides MySQL connectivity using [mysql2](https://www.npmjs.com/package/mysql2).

## Installation

```bash
npm install @gradii/fedaco @gradii/fedaco-mysql-driver mysql2
# or
pnpm add @gradii/fedaco @gradii/fedaco-mysql-driver mysql2
# or
yarn add @gradii/fedaco @gradii/fedaco-mysql-driver mysql2
```
