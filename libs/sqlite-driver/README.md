# @gradii/fedaco-sqlite-driver

SQLite driver for [Fedaco ORM](https://www.npmjs.com/package/@gradii/fedaco). Supports both [better-sqlite3](https://www.npmjs.com/package/better-sqlite3) and [sqlite3](https://www.npmjs.com/package/sqlite3).

## Installation

```bash
# Using better-sqlite3 (recommended)
npm install @gradii/fedaco @gradii/fedaco-sqlite-driver better-sqlite3
# or
pnpm add @gradii/fedaco @gradii/fedaco-sqlite-driver better-sqlite3
# or
yarn add @gradii/fedaco @gradii/fedaco-sqlite-driver better-sqlite3
```

```bash
# Using sqlite3
npm install @gradii/fedaco @gradii/fedaco-sqlite-driver sqlite3
# or
pnpm add @gradii/fedaco @gradii/fedaco-sqlite-driver sqlite3
# or
yarn add @gradii/fedaco @gradii/fedaco-sqlite-driver sqlite3
```
