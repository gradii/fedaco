
# Getting Started

## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) version 18 or higher.

::: code-group

```sh [npm]
$ npm install @gradii/fedaco
```

```sh [yarn]
$ yarn add @gradii/fedaco
```
:::

### Examples Repo
There are fedaco examples repo can found [here](https://github.com/gradii/fedaco-examples)
